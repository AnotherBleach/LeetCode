> 题目总结：
>
> * 重复数字中求单独出现的数字
>
> 往往可以利用数字的二进制位特点解决，一般来说，异或用的比较多
>
>   - A xor A = 0
>   - A xor 0 = A
>
> 这个[链接](https://www.cnblogs.com/Kobe10/p/6306183.html)里总结了很多重复数字的题型
>
>   * 每个数字出现了2次，除了一个数字，找出来。(异或)
>   * 每个数字出现了3次，除了一个数字，找出来（对3求余数）
>   * ....
>   * 2222222333333333
